Halaman PNM (blank)

keterangan: pada target dengan tipe blank, saat kiata membuka link yang ada pada tag a, maka browser akan membuka link tersebut melalui tab yang berbeda, hal tersebut membuat kita tidak dapat kembali membuka halaman awal ketika kita menekan tombol back

Halaman list (parent)

keterangan: Pada target tipe parent, ketika kita membuka link pada tag a tersebut, kita akan otomatis diarahkan menuju link tersebut di frame parent

Halaman paragraf (self)

keterangan: Pada target tipe self, ketika kita membuka link pada tag a tersebut, kita akan otomatis diarahkan menuju link tersebut tanpa membuka tab baru. Jika kita tidak menentukan atribut target, link akan secara otomatis terbuka di tab atau jendela yang sama.

Halaman Heading

keterangan: Pada target tipe slef, ketika kita membuka link pada tag a tersebut, kita akan otomatis membuka halaman website tersebut dan membatalkan semua frame.
